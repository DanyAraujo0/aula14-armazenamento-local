Este repositório contém três exercícios práticos demonstrando as diferentes formas de armazenar dados no navegador do usuário usando JavaScript.

### 1. Bloco de Notas (`localStorage`)
* `localStorage` (Armazenamento permanente). O texto é salvo em tempo real enquanto o usuário digita. Ao fechar e reabrir o navegador, as anotações continuam lá.

### 2. Carrinho de Compras (`sessionStorage`)
* `sessionStorage` (Armazenamento temporário). Os dados sobrevivem a atualizações de página (F5), mas são completamente apagados assim que a aba do navegador é fechada.

### 3. Preferência de Idioma (`Cookies`)
* `document.cookie` (Armazenamento com data de expiração). O idioma é salvo em um cookie com validade exata de 14 dias. O usuário também tem a opção de remover essa preferência a qualquer momento.
